# Privacy Policy

Timmeny-ToDo-OS is a private personal workflow tool used to create action items in configured Monday.com boards.

## Information Sent

When you use a GPT Action connected to Timmeny-ToDo-OS, the GPT may send:

- The todo title you ask it to create
- The selected destination list, such as `todo` or `gs`

## How Information Is Used

The Timmeny-ToDo-OS API uses this information to create an item in the configured Monday.com board. It sends the todo title and destination information to Monday.com through the Monday.com API.

## Storage

Timmeny-ToDo-OS does not intentionally store todo request data in its own database. Request and response metadata may appear in Railway service logs, and created items are stored in Monday.com according to Monday.com's own policies.

## Sensitive Information

Do not submit sensitive personal, financial, medical, legal, or confidential information as todo titles.

## Third Parties

This workflow uses:

- Railway, to host the Timmeny-ToDo-OS API
- Monday.com, to store created todo items
- OpenAI ChatGPT, when using the GPT Action interface

## Contact

For questions about this policy, contact the repository owner.
